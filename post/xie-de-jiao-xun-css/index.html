血的教训，震惊，不能漏掉css属性

2019-07-19

如何实现长度不够时的...效果，可以用这一套三连属性来实现

overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;

然而千万不能写漏掉了。 在普通的样式中，长度不够时我们很容易发现错误的实现，然而较复杂的就需要我们仔细检查。example:

实现这个的时候 我们前面的属性需要用到...的效果，这时候左边长度不固定，测试的时候有时候不一定会想到就是缺少属性引起的BUG，这题我用到的具体如下： demo

.box{
		overflow: hidden;
		height: 32px;
}
.title{
		font-size: 26px;
		height: 32px;
		line-height: 32px;
		overflow: hidden;
		white-space: nowrap;
		text-overflow: ellipsis;
		vertical-align: middle;
		text-align: left;
}
.tag-box{
		float: right;
		height: 32px;
		line-height: 32px;
		vertical-align: middle;
		background: inherit;
}

.tag{
		display: inline-block;
		border: 1px solid #2c94e9;
		color: #2c94e9;
		height: 32px;
		line-height: 32px;
		border-radius: 5px;
		box-sizing: border-box;
		font-size: 16px;
		padding: 0 2px;
}

<div class = 'box'>
		<div class="tag-box">
				<div class='tag'>连载</div>
				<div class='tag'>连载</div>
				<div class='tag'>连载</div>
				<div class='tag'>连载</div>
		</div>
		<div class="title">这次是一个新的故事这次是一个新的故事这次是一个新的故事</div>
</div>

写的时候title里面漏掉了over-flow:hidden; 所以未能实现效果，然后调试了很久才恍然大悟。所以记下来提醒自己不能再犯这种错误
